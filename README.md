# Student Management System

A Laravel 13 Student Management System developed using Laravel Breeze, Inertia.js, React, and the Facade design pattern.


# Features
- User registration and login
- Authenticated dashboard
- Student CRUD operations
  - Add new students
  - View student list
  - Edit student details
  - Delete students
- Student image upload
- Student status management
  - Active
  - Inactive
- Dashboard overview cards
- Generate report as PDF
- Export student data as Excel
- Notification and validation messages
- Responsive user interface
- Search Student Using Id, Name or Age 


## Technologies Used
- Laravel 13
- PHP
- Composer
- Laravel Breeze
- Inertia.js
- React
- Vite
- MySQL
- Tailwind CSS / Bootstrap
- Facade Design Pattern
